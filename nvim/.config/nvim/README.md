# Neovim

Neovim config using [LazyVim](https://github.com/LazyVim/LazyVim)

## 📂 File Structure

<pre>
~/.config/nvim
├── lua
│   ├── config
│   │   ├── autocmds.lua
│   │   ├── keymaps.lua
│   │   ├── lazy.lua
│   │   └── options.lua
│   └── plugins
│       └── example.lua
├── init.lua
└── stylua.toml
</pre>

# Remember Me

If an update breaks try deleting `~/.local/share/nvim/` and `$HOME/.cache/nvim/`
